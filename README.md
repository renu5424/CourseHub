# CourseHub - Your AI-Powered Learning Platform

Welcome to CourseHub! This is a Next.js application built with Firebase and Genkit, designed to be an AI-powered platform for creating and taking online courses.

## Getting Started

This project is ready to be developed locally. Follow the steps below to get it running on your machine.

### Prerequisites

- Node.js (v18 or later recommended)
- npm or another package manager

### Installation

1.  **Install dependencies:**
    Open your terminal in the project root and run the following command to install all the necessary packages.

    ```bash
    npm install
    ```

2.  **Run the development server:**
    Once the installation is complete, you can start the Next.js development server.

    ```bash
    npm run dev
    ```

The application will now be running locally, typically at `http://localhost:3000`. You can open this URL in your browser to see your application.

## Key Technologies

- **Next.js**: React framework for building the user interface.
- **Firebase**: Provides the backend services, including Authentication and Firestore for the database.
- **Genkit**: Powers the AI features, such as the AI Tutor and content generation.
- **ShadCN/UI**: Used for the component library to build the user interface.
- **Tailwind CSS**: For styling the application.
