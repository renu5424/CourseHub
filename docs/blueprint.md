# **App Name**: CourseHub

## Core Features:

- AI Lesson Generation: Use AI to suggest lesson topics based on the course title and generate lesson content.
- Role-Based Authentication: Implement user authentication with 'Mentor' and 'Student' roles.
- Course Management (CRUD): Enable mentors to create, read, update, and delete courses.
- Lesson Management: Allow mentors to add, delete, and reorder lessons within a course.
- Student Dashboard: Display enrolled and completed courses for students.
- AI Quiz Generation: Generate multiple-choice quizzes based on lesson content with varying difficulty levels.
- Diksha AI Tutor Tool: Integrate a conversational AI chat interface (Diksha) for answering general questions and providing lesson help. The tutor reasons to choose different explanation methods (storytelling, simplification) to address studentsâ€™ specific needs.
- Firestore Integration: Store user profiles, course information, lessons, and enrollment details in Firestore.
- Minimal Security: Grant access to mentor and student use with minimal security.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to inspire trust and knowledge.
- Background color: Light purple (#E1BEE7) for a clean, uncluttered learning environment.
- Accent color: Bright purple (#9C27B0) to highlight key actions and progress.
- Body and headline font: 'Inter', a sans-serif for a clean and modern feel.
- Use modern, outline-style icons from ShadCN UI to represent course categories and actions.
- Utilize a card-based layout with clear information hierarchy on dashboards and course catalogs.
- Implement subtle transitions and animations on UI elements.
